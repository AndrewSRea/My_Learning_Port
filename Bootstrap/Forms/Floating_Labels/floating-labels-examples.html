Simple floating labels within form fields using the .form-floating class

With a value already defined in the <input> element, the <label> element will automatically adjust to its floated position

Form validation styles also work with the .form-floating class

By default, <textarea>s with the class .form-control will be the same height as <input>s.

To set a custom height on the <textarea>, do not use the rows attribute. Instead, set an explicit height (either inline or via custom CSS).
