Bootstrap Position Utilities Examples

Position Utilities Example 01

Arrange elements easily with the edge positioning utilities. The format is {property}-{position}.

Position Utilities Example 02

Elements can be centered with the transform utility class .translate-middle. This class applies the transformations translateX(-50%) and translateY(-50%) to the element which, in combination with the edge positioning utilities, allows you to absolute center an element.

Position Utilities Example 03

By adding .translate-middle-x or .translate-middle-y classes, elements can be positioned only in horizontal or vertical direction.
