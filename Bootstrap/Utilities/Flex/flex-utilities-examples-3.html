Flex Utilities Example 08

Use .flex-grow-* utilities to toggle a flex item's ability to grow to fill available space. In this example, the .flex-grow-1 element uses all available space it can, while allowing the remaining two flex items their necessary space.

Flex Utilities Example 09

Use .flex-shrink-* utilities to toggle a flex item's ability to shrink, if necessary. In this example, the second flex item with .flex-shrink-1 is forced to wrap its contents to a new line, "shrinking" to allow more space for the previous flex item with .w-100.
