Bootstrap Display Utilities Code Examples

Display Utility Example: 01

Change the value of the display property with Bootstrap's responsive display utility classes. These classes are named using the format: .d-{value} for the breakpoint xs; .d-{breakpoint}-{value} for breakpoints sm, md, lg, xl, and xxl.
value is the display value, such as inline or block in the examples below.

d-inline
d-inline
d-block d-block

Display Utility Example: 02

Use responsive display classes for showing and hiding elements by device. To hide elements, simply use the .d-none class or one of the .d-{sm,md,lg,xl,xxl}-none classes for any responsive screen variation. To show an element only on a given interval of screen sizes, you can combine one .d-*-none class with a .d-*-* class. For example, .d-none .d-md-block .d-xl-none .d-xxl-none will hide the element for all screen sizes except on medium and large devices.
Adjust the width of your browser to see the effects of the responsive display classes.

hide on lg and wider screens
hide on screens smaller than lg
