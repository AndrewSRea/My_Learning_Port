Bootstrap Sizing Utilities Examples

Sizing Utilities Example 01

Width and height utilities are generated from Bootstrap's utility API in _utilities.scss. Includes support for 25%, 50%, 75%, 100%, and auto by default. Modify those values as you need to generate different utilities here.
