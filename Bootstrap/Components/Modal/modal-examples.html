Bootstrap Modal Component Examples

Modal Component Example 01

This modal example has its position and display overridden by Bootstrap's CSS. Included in this modal are the modal header (with a dismiss action), modal body, and modal footer. It is also able to be toggled by clicking the button below. The modal will slide down and fade in from the top of the page.

Modal title

Woohoo, you're reading this text in a modal!

Modal Component Example 02

When the backdrop of a modal is set to static, the modal will not close when clicking outside of it.

Modal title

I will not close if you click outside me. Don't event try to press escape key.

Modal Component Example 03
