Bootstrap Carousel Component Examples

Carousel Component Example 01

A carousel with slides only. Note the presence of the .d-block and .w-100 utility classes on the carousel images to prevent browser default image alignment.

Carousel Component Example 02

Adding in previous and next controls, using <button> elements.

Adding indicators to the carousel, along with the controls.

Carousel Component Example 05

Adding the .carousel-fade class to the carousel in order to animate slides with a fade transition.

Carousel Component Example 06

Adding data-bs-interval="" to a .carousel-item class to change the amount of time to delay between automatically cycling to the next slide.

Carousel Component Example 07

To disable touchscreen swiping, the data-bs-touch attribute can be used with a value of false. Also, the data-bs-ride atribute is excluded, and data-bs-interval="false" is used so the carousel doesn't autoplay.

Add .carousel-dark to the .carousel for darker controls, indicators, and captions.
