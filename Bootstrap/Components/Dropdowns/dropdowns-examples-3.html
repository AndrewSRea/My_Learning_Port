Bootstrap Dropdowns Component Examples - Page 3

Dropdowns Component Example 14

Dropdown menus are automatically positioned 100% from the top and along the left side of their parents. This can be changed with the directional .drop* classes. In this example, add a .dropdown-menu-end class to the .dropdown-menu class to right-align the dropdown menu.

Dropdowns Component Example 15

To use responsive alignment, disable dynamic positioning by adding the data-bs-display="static" attribute and use the responsive variation classes. To align right a dropdown menu with a given breakpoint or larger, add .dropdown-menu{-sm|-md|-lg|-xl|-xxl}-end.

Dropdowns Component Example 16

To align left a dropdown menu with a given breakpoint or larger, add .dropdown-menu-end and .dropdown-menu{-sm|-md|-lg|-xl|-xxl}-start.
