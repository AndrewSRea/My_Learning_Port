A large block of connected links is used for pagination, making links hard to miss and easily scalable, while providing large hit areas. It is built with HTML list elements so scren readers can announce the number of available links. A wrapping <nav> identifies the block as a navigation section. The aria-label="" provides a description for the <nav> to reflect its purpose.
