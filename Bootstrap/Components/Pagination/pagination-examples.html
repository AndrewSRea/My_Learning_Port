Bootstrap Pagination Component Examples

Pagination Component Example 01

A large block of connected links is used for pagination, making links hard to miss and easily scalable, while providing large hit areas. It is built with HTML list elements so screen readers can announce the number of available links. A wrapping <nav> identifies the block as a navigation section. The aria-label="" provides a description for the <nav> to reflect its purpose.

Pagination Component Example 03

Pagination links are customizable for different circumstances. Use .disabled links which appear un-clickable, and .active to indicate the current page. Always add tabindex="-1" on disabled links.
