Navs and Tabs Component Example 14

Dynamic tabbed interfaces require role="tablist", role="tab", role="tabpanel", and additional aria- attributes in order to convey their structure, functionality, and current state to users of assistive technologies. It's recommended to use <button> elements for the tabs, like this example and the one above. However, if you're using <nav>, you shouldn't add role="tablist" directly to it, as this would override the element's native role as a navigation landmark. Instead, switch to an alternative element, like a <div>, and wrap the <nav> around it.
