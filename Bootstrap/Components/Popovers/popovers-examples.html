Bootstrap Popovers Component Examples

Popovers Component Example 01

Popover components rely on the third-party Popper.js for positioning, which is included in Bootstrap's bootstrap.bundle.min.js. This example adds a popover toggle on a button with the data-bs-toggle="popover" attribute.

Popovers Component Example 02

Four options of popovers are available: top, right, bottom, and left aligned.

Popovers Component Example 03

Use the focus trigger to dismiss popovers on the user's next click of a different element than the toggle element.

Popovers Component Example 04

Elements with the disabled attribute aren't interactive, meaning users cannot hover or click them to trigger a popover. As a workaround, trigger the popover from a wrapper <div> or <span>, and make it keyboard-focusable using tabindex="0".
